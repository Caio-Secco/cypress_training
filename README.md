# Cypress Test Automation

This project contains end-to-end (E2E) automated tests built with Cypress and JavaScript.
The automation focuses on validating main user flows and ensuring application quality.

## Requirements

Before running the project, make sure you have installed:

- Node.js

## Installation

Run the following command in the project root:

```bash
npm install
````
## Running the Tests

### Open Cypress UI (interactive mode)

```bash
npx cypress open
```
### Run tests via command line (headless mode)
```bash
npx cypress run
```
## Project Overview

- End-to-end automated tests
- Page Object Model (POM)
- Use of fixtures for test data
- Clean and maintainable structure
- Focus on user flows and business rules
